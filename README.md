ðŸ§  Queries and Their Meanings

1. Display All Records
SELECT * FROM pizza_sales;
Shows all rows and columns in the table for exploration.

2. Total Revenue
SELECT SUM(total_price) AS Total_Revenue FROM pizza_sales;
Calculates the total revenue generated from all pizza sales.

3. Average Order Value
SELECT (SUM(total_price)/COUNT(DISTINCT order_id)) AS avg_order_value FROM pizza_sales;
Finds the average amount spent per order.

4. Total Pizzas Sold
SELECT SUM(quantity) AS Total_Pizza_sold FROM pizza_sales;
Returns the total number of pizzas sold.

5. Total Orders
SELECT COUNT(DISTINCT order_id) AS Total_orders FROM pizza_sales;
Counts the number of unique customer orders.

6. Highest Price Pizzas
SELECT TOP 5 pizza_name, total_price 
FROM pizza_sales 
ORDER BY total_price DESC;
Displays the 5 most expensive pizza sales.

7. Unique Pizza Categories
SELECT DISTINCT pizza_category FROM pizza_sales;
Lists all distinct pizza categories.

8. Filter: Classic Category
SELECT * FROM pizza_sales WHERE pizza_category = 'Classic';
Retrieves only Classic category pizzas.

9. Top 10 Highest-Value Sales
SELECT TOP 10 * FROM pizza_sales ORDER BY total_price DESC;
Shows the top 10 most expensive transactions.

10. Total Sales by Category
SELECT pizza_category, SUM(total_price) AS total_sales
FROM pizza_sales
GROUP BY pizza_category;
Calculates revenue generated by each pizza category.

11. Average Price per Category
SELECT pizza_category, AVG(unit_price) AS avg_price
FROM pizza_sales
GROUP BY pizza_category;
Finds the average price of pizzas in each category.

12. Total Quantity Sold per Category
SELECT pizza_category, SUM(quantity) AS total_quantity
FROM pizza_sales
GROUP BY pizza_category;
Shows total pizzas sold per category.

13. Categories With Sales Greater Than 1 Lakh
SELECT pizza_category, SUM(total_price) AS total_sales
FROM pizza_sales
GROUP BY pizza_category
HAVING SUM(total_price) > 100000;
Filters high-performing categories based on revenue.

14. Orders in January 2015
SELECT * FROM pizza_sales
WHERE order_date BETWEEN '2015-01-01' AND '2015-01-31';
Retrieves all orders placed in January 2015.

15. Pizzas With "Cheese" in the Name
SELECT * FROM pizza_sales
WHERE pizza_name LIKE '%Cheese%';
Finds all pizzas that contain the word "Cheese" in their name.
